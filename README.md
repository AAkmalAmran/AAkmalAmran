- **Sistem Restoran (Python + Streamlit)** – Aplikasi manajemen restoran terintegrasi dengan basis data MySQL.
- **Game Text-Based (Java + MySQL)** – Pengembangan logika permainan menggunakan konsep Pemrograman Berbasis Objek.
